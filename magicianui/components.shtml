组件

无数可复用的组件，包括菜单、列表、面包屑等更多功能。

统一样式

组件样式，我们使用了 MComponents.css。

导航

常规导航

导航组件都依赖同一个 .nav 类，状态类也是共用的。改变修饰类可以改变样式。

  • 菜单1
  • 菜单2
  • 菜单3

<ul class="nav">
  <li><a href="#">菜单1</a></li>
  <li><a href="#">菜单2</a></li>
  <li><a href="#">菜单3</a></li>
</ul>

另外，还提供了 .nav_tab 、 .nav_pills 、 .sidebar_nav 类，为的是给导航赋予相应的样式

标签导航

HTML 标记相同，但使用 .nav_tab 类：

  • 菜单1
  • 菜单2
  • 菜单3

<ul class="nav nav_tab">
  <li class="current"><a href="#">菜单1</a></li>
  <li><a href="#">菜单2</a></li>
  <li><a href="#">菜单3</a></li>
</ul>

胶囊式导航

HTML 标记相同，但使用 .nav_pills 类：

  • 菜单1
  • 菜单2
  • 菜单3

<ul class="nav nav_pills">
  <li class="current"><a href="#">菜单1</a></li>
  <li><a href="#">菜单2</a></li>
  <li><a href="#">菜单3</a></li>
</ul>

竖排导航

HTML 标记相同，但使用 .sidebar_nav 类：

  • 菜单1
  • 菜单2
  • 菜单3

<ul class="nav sidebar_nav">
  <li class="current"><a href="#">菜单1</a></li>
  <li><a href="#">菜单2</a></li>
  <li><a href="#">菜单3</a></li>
</ul>

面包屑

在一个带有层次的导航结构中标明当前页面的位置。

首页 > nav > nav

<div class="breadcrumbs"><a href="#">首页</a> > <a href="#">nav</a> > <span class="current">nav</span></div>

标签

列表

常见的3种形态的列表

简单新闻列表

HTML 标记相同，但使用 .news_list 类：

  • 2016-10-10title
  • 2016-10-10title

<ul class="news_list">
  <li><a href="#"><span class="time">2016-10-10</span>title</a></li>
  <li><a href="#"><span class="time">2016-10-10</span>title</a></li>
</ul>

摘要列表

HTML 标记相同，但使用 .news_list 类，在li里面添加了div.title

  • 标题2016-10-10

    概要

  • 标题2016-10-10

    概要


<ul class="news_list">
  <li>
    <div class="title"><a href="#">标题</a><span class="time">2016-10-10</span></div>
    <p>概要</p>
  </li>
</ul>
